Machine Learning

Machine Learning adalah cabang dari Artificial Intelligence (AI) yang memungkinkan komputer belajar dari data tanpa diprogram secara langsung. Dengan machine learning, sistem dapat mengenali pola, membuat prediksi, dan meningkatkan kinerja secara otomatis dari pengalaman data sebelumnya.

Supervised Learning

Supervised Learning adalah metode machine learning yang menggunakan data yang sudah memiliki label. Model dilatih menggunakan pasangan data input dan output yang benar sehingga dapat memprediksi hasil untuk data baru. Contoh algoritma supervised learning adalah Linear Regression, Decision Tree, dan Support Vector Machine.

Unsupervised Learning

Unsupervised Learning adalah metode machine learning yang digunakan untuk menemukan pola atau struktur dalam data yang tidak memiliki label. Model akan mencoba mengelompokkan atau menemukan hubungan antar data secara otomatis. Contoh algoritma unsupervised learning adalah K-Means Clustering dan Hierarchical Clustering.
